# Bulls Asset Manager

**Bulls Asset Manager** is a Blender addon for quickly rendering asset previews and building organized asset libraries — with direct integration to the [PolyAssetVault](https://www.polyassetvault.com) marketplace.

---

## What it does

- **Render Previews** — Generate thumbnails for your assets using customizable light setups and HDRI environments, directly inside Blender
- **Asset Library** — Organize and manage your assets using Blender's native Asset Browser
- **Marketplace** — Browse, download, and upload assets directly to PolyAssetVault from within Blender
