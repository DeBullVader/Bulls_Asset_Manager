# Installation

## Requirements

- Blender **4.2 or newer**
- A [PolyAssetVault](https://www.polyassetvault.com) account (for marketplace features)

---

## Install the Addon

1. Download the latest release from [GitHub](https://github.com/DeBullVader/Bulls_Asset_Manager/releases)
2. Open Blender and go to **Edit → Preferences → Add-ons**
3. Click **Install...** and select the downloaded `.zip` file
4. Enable the addon by checking the checkbox next to **Bulls Asset Manager**

---

## Verify Installation

Once installed, open the **N-Panel** in the 3D Viewport (press `N`) and look for the **BullTools** tab.
