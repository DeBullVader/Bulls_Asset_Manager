# First Setup

After installing the addon, follow these steps to get everything configured.

---

## 1. Set Your Asset Library Path

1. Go to **Edit → Preferences → Add-ons → Bulls Asset Manager**
2. Under **Library Path**, set the folder where your asset `.blend` files are stored
3. Blender will register this folder as an Asset Library automatically

---

## 2. Set Your Author Name

In the same preferences panel, enter your name under **Author**. This will be used as the default author when marking assets.

---

## 3. Start Rendering

That's all the setup needed. Select your assets in the viewport, open the **N-Panel → BullTools → Asset Manager**, and click **Render Previews**. The addon automatically handles the render scene — no manual setup required.
