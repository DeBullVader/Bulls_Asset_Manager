# PolyAssetVault Marketplace

[PolyAssetVault](https://www.polyassetvault.com) is a marketplace for Blender assets. Bulls Asset Manager includes direct marketplace integration so you can browse, download, and upload assets without leaving Blender.

---

## Features

| Feature | Description |
|---|---|
| **Login** | Sign in to your PolyAssetVault account from within Blender |
| **Profile** | View your account and purchased assets |
| **Browse** | Browse assets available on the marketplace |
| **Download** | Download purchased assets directly into your asset library |
| **Upload** | Upload your own assets to the marketplace |
